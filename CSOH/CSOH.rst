CSOH
--------
Cassini State Obliquity Histogram.
This plotting routine allows a user to replicate Figure 2 of the associated paper, which shows the initial and final obliquity distribution in the simulation set.
 

To run this example
-------------------

.. code-block:: bash
    python CSOH.py <pdf | png>
